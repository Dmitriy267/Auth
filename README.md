# Реализация простой формы авторизации на React

Цель данного проекта- показать простую реализацию авторизации.
Используется две формы: форма введения логина и регистрация. Используются два хука 'useState' и 'useReducer'.
Это простой базовый пример без дополнительных хуков, без валидации и без сохранения данных, например, в 'localStorage'.
